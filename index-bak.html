公司產品

先進的創意 BAUDIO BS-265

BAUDIO BS-265將藍芽 / 擴大器 / 吸頂喇叭音箱三合一完美結合，全部安裝於天花板，或鑲崁於牆壁上，不但省下傳統擴大器、喇叭音箱及音響櫃所占空間，且仍保有中型音箱喇叭的音質表現。

外觀上更完全看不出任何線材，型塑高品味的居家時尚簡約風格。
音源可與家裡所有手機 / 平板 / 筆電 / PC等具有藍牙功能的裝置連結，享受無拘無束無線傳輸音樂的樂趣；聽歌、背景音樂、看電影、聽網路收音機、上Youtube、打電動、學語言…

走到哪聽到哪，讓家庭娛樂中心有了全新的定義。

產品特色

  • 世界第一台專屬帳號及密碼設計---可避免附近的藍牙發射裝置干擾。

  • 藍牙 / 擴大器 / 吸頂喇叭三合一設計--不占空間，不需音響櫃，不需考慮隱藏拉線問題，外觀上完全看不到任何線材。

  • 安裝容易---只有一條電源線及一條喇叭線，容易DIY。

  • 防水防潮設計---適用於浴室、廚房或室外等暴露於有水氣的環境中。

  • 隨開隨用---不需充電，打開電燈即同步啟動電源，隨時可賞聆音樂。

  • 來電時音樂自動靜音。

產品規格

  • 藍牙
    格式 3.0 A2DP AVRCP profile support
    有效距離 10~15 公尺
  • 喇叭
    單體 3/4 吋高音 / 6.5 吋中低音
    效率 93dB@1w/1m
